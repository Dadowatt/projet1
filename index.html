Présentation de la formation

Devenez Développeur Web grâce à une formation complète

La formation DevSchool est destinée aux personnes souhaitant découvrir le développement web, même sans expérience préalable. Elle a été conçue pour transmettre les bases essentielles du web moderne à travers une pédagogie simple, progressive et orientée vers la pratique. Tout au long de la formation, les apprenants apprennent à structurer des pages web avec HTML5, à les mettre en forme avec CSS et à comprendre les bases de JavaScript. L'objectif est de rendre les participants autonomes et capables de réaliser des projets web concrets.

Programme de la formation

HTML

Ce module permet d'apprendre à structurer une page web à l'aide du langage HTML5. Les apprenants découvrent les balises sémantiques, la hiérarchie du contenu et les bonnes pratiques d'accessibilité.

CSS

Le module CSS est dédié à la mise en forme et au design des pages web. Il permet d'apprendre à créer des interfaces modernes, claires et adaptées à tous les écrans.

Javascript

Ce module introduit les bases de JavaScript afin de rendre les pages web interactives. Les apprenants découvrent la logique de programmation et les premières interactions utilisateur.
